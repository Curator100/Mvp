# WSEP - Women Safety & Empowerment Program

## 🚀 Deploy করুন

### ১. Supabase Setup
`schema.sql` ফাইলটি Supabase SQL Editor-এ রান করুন।

### ২. GitHub → Netlify Deploy
1. সব ফাইল GitHub-এ push করুন
2. Netlify-তে নতুন সাইট তৈরি করুন
3. GitHub repo সংযুক্ত করুন
4. Deploy!
